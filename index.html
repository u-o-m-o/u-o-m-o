An User oriented Social Media Ontology

Uomo Ontology describes how social media are structured. Its aim is to cover all the features that characterize social media, with a broad focus on both the most famous Social Media and less common ones. From our perspective, all social media are based on the fact that there are users who interact through the social media platform. Without users, Social media would not have any significance. Starting from this basic concept we decided to create a user oriented ontology. After a detailed analysis of several social media, we found out that there are 2 main elements that all social media have in common: users’ profiles and a set of interactions. The latter can be divided into interactions with the platform, which generate contents, and interactions with other users, which generate networks.

U.O.M.O. PROFILE

The profile is a place that can be customized with users' information. They can be both personal information or facts related to the user’s interests. The level of details provided and their nature depend on the social media.

W.I.P.

U.O.M.O. CONTENT

This section of the ontology describes the interactions that users have with the platform, including uploading and sharing contents such as posts and media.

W.I.P.

U.O.M.O. NETWORK

The main purpose of social media is to let users getting in touch with other users (whether they’re friends or not). Users want to make visible their social networks or to start new networking. All the possible declinations of this type of interactions are described in this section of the ontology.
